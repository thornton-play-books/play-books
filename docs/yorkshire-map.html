Mapping Alice Thornton’s Books: Yorkshire

Author

Sharon Howard

This map and linked data are early work in progress for the Alice Thornton’s Books project. The map shows places in Yorkshire that were mentioned by Alice Thornton in her Books.

Red pins mark places where Thornton lived for significant periods. Hover over pins to see place name labels. The table below the map provides links to related material in a number of online resources.

To filter place names click/tap in the Find places box and select from the drop down menu (or start typing names and hit enter to autocomplete). Multiple selections are allowed.

Or click on the make a selection button in the map (just below the + and - buttons) to narrow down to a smaller area of the map.

(You can also highlight a single place by clicking on the place name in the table (click again to reset) but note that a limitation of the map is that it will not automatically move to show the highlighted place.)
